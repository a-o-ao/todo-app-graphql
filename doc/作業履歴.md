# 作業メモ

## 環境構築

```shell
# NestJS 関連
npm install -g @nestjs/cli

nest new backend
cd backend

# GraphQL 関連
npm install @nestjs/graphql @nestjs/apollo graphql apollo-server-express

# Prisma 関連
npm install prisma --save-dev
npm install @prisma/client

npx prisma init

# マイグレーション
# npx prisma migrate dev --name init

# DB取り込み
# (schema.prismaにスキーマ定義が追加される)
npx prisma db pull

# Prisma クライアント生成
npx prisma generate
```


## NestJSモジュール作成

-  前提：Prisma に Todo モデルがある（npx prisma db pull or schema.prisma に定義済）

1. TodoModule を作成
```shell
nest g module todo
nest g service todo
nest g resolver todo
```

2. GraphQL用 DTO / Input を作成
- src/todo/dto/create-todo.input.ts
- src/todo/dto/update-todo.input.ts

3. GraphQL用 ObjectType を作成
- src/todo/entities/todo.entity.ts

4. サービスにCRUDロジックを書く
- src/todo/todo.service.ts

5. GraphQLリゾルバを作成
- src/todo/todo.resolver.ts

6. AppModule に TodoModule を追加
